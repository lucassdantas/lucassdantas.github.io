Política de Privacidade – Pega a Visão

Última atualização: 03 de maio de 2026

A sua privacidade é importante para nós. Esta Política de Privacidade descreve como o aplicativo Pega a Visão coleta, usa e protege as informações dos usuários.

1. Informações que coletamos

1.1 Informações fornecidas pelo usuário

  • Nome (se aplicável)
  • E-mail (caso haja autenticação)
  • Informações inseridas em denúncias

1.2 Dados de localização

Coletamos a localização do usuário para:

  • Exibir incidentes próximos
  • Permitir o envio de denúncias com localização

A localização é utilizada apenas enquanto o aplicativo está em uso.

1.3 Imagens e câmera

O aplicativo pode acessar a câmera e a galeria para:

  • Permitir o envio de fotos relacionadas a denúncias

As imagens são enviadas para armazenamento seguro e não são acessadas sem ação do usuário.

1.4 Dados de uso

Podemos coletar dados básicos de uso para melhorar a experiência, como:

  • Interações com o app
  • Erros e falhas

2. Como usamos as informações

  • Permitir o funcionamento das denúncias
  • Exibir conteúdo relevante baseado na localização
  • Melhorar a experiência do usuário
  • Garantir segurança e integridade da plataforma

3. Armazenamento de dados

Os dados são armazenados utilizando serviços de terceiros confiáveis, como o Supabase.

  • As imagens enviadas são armazenadas em buckets seguros
  • Dados podem ser protegidos por autenticação e regras de acesso

4. Compartilhamento de dados

Não vendemos dados pessoais.

Os dados podem ser compartilhados apenas quando necessário para:

  • Funcionamento do aplicativo
  • Cumprimento de obrigações legais

5. Publicidade

O aplicativo pode exibir anúncios de terceiros.

Esses serviços podem coletar informações como:

  • Identificadores do dispositivo
  • Dados de uso

6. Segurança

Adotamos medidas para proteger os dados dos usuários, incluindo:

  • Comunicação segura (HTTPS)
  • Controle de acesso aos dados
  • Armazenamento em serviços confiáveis

7. Direitos do usuário

  • Solicitar a exclusão dos seus dados
  • Solicitar acesso às suas informações

8. Retenção de dados

Os dados são mantidos apenas pelo tempo necessário para cumprir as finalidades descritas nesta política.

9. Alterações nesta política

Esta política pode ser atualizada periodicamente. Recomendamos que você revise esta página regularmente.

10. Contato

Se tiver dúvidas sobre esta Política de Privacidade, entre em contato:

Email: SEUEMAIL@AQUI.COM

11. Legislação aplicável

Esta política é regida pelas leis do Brasil, incluindo a Lei Geral de Proteção de Dados (LGPD).

Ao utilizar o aplicativo Pega a Visão, você concorda com esta Política de Privacidade.
