Política de Privacidade

Última atualização: 2026

Esta Política de Privacidade descreve como o aplicativo Mensagem do Dia coleta, usa, armazena e protege os dados dos usuários, em conformidade com as políticas do Google Play.

2. Coleta e Uso de Dados

O aplicativo não coleta dados pessoais identificáveis, como nome, e-mail, telefone ou localização.

No entanto, alguns dados podem ser coletados automaticamente por serviços de terceiros, incluindo:

  • Identificadores de publicidade (Advertising ID)
  • Informações do dispositivo (modelo, sistema operacional)
  • Dados de uso e interação com anúncios

Esses dados são utilizados para:

  • Exibição de anúncios
  • Melhoria da experiência do usuário
  • Funcionamento adequado dos serviços integrados

3. Serviços de Terceiros

O aplicativo utiliza serviços de terceiros, como o Google AdMob, que podem coletar dados automaticamente para personalização e exibição de anúncios.

Esses serviços operam de acordo com suas próprias políticas de privacidade. Para mais informações, acesse:
https://policies.google.com/privacy

4. Compartilhamento de Dados

O aplicativo não vende, aluga ou compartilha dados pessoais dos usuários.

Dados não pessoais podem ser processados por serviços de terceiros apenas para fins operacionais, como exibição de anúncios.

5. Armazenamento e Retenção de Dados

O aplicativo utiliza armazenamento local no dispositivo para:

  • Salvar a mensagem gerada no dia
  • Controlar a geração de uma mensagem por dia

Esses dados:

  • Permanecem apenas no dispositivo do usuário
  • Não são enviados para servidores externos
  • Podem ser removidos ao limpar os dados do aplicativo

6. Segurança dos Dados

Adotamos medidas razoáveis para proteger as informações. Dados tratados por serviços de terceiros utilizam protocolos seguros, como HTTPS.

7. Permissões

O aplicativo não solicita permissões sensíveis como localização, câmera, microfone, contatos ou armazenamento externo.

8. Consentimento

Ao utilizar o aplicativo, o usuário concorda com esta Política de Privacidade.

9. Alterações nesta Política

Esta Política pode ser atualizada periodicamente. Recomendamos a revisão regular desta página.

10. Conformidade

O aplicativo busca cumprir as leis aplicáveis de proteção de dados e as políticas do Google Play, incluindo transparência, segurança e limitação de uso de dados.
